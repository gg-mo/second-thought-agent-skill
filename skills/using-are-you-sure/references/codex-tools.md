# Codex Tool Mapping

| Skill references | Codex equivalent |
|-----------------|------------------|
| `Skill` tool | Native skills discovery |
| `TodoWrite` | `update_plan` |
| `Task` | `spawn_agent` (if enabled) |
| `Read/Write/Edit/Bash` | Native Codex file and shell tools |

## Notes

- Codex discovers skills from `~/.agents/skills/`.
- For this repo, install via symlink from `.codex/INSTALL.md`.
