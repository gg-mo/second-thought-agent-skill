# Are You Sure API Contract v1

## Input

Required fields:
- `original_intent`
- `current_context`
- `proposal_type`
- `proposal`
- `rationale`

Optional fields:
- `constraints`
- `risk_level`
- `stage`
- `should_challenge`
- `mode`
- `explainability`
- `reversibility`
- `estimated_cost`
- `blast_radius`

## Output

- `status`
- `summary`
- `goal_alignment`
- `concerns`
- `assumptions`
- `better_options`
- `challenge_prompt`
- `recommended_next_step`
- `prompt_to_human`
- `confidence`
- `decision_factors`

## Compatibility

v1 follows additive compatibility:
- Existing fields are stable.
- New optional fields may be added.
- Enum narrowing is a breaking change.
